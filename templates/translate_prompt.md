## 翻訳ルール

### 文体・表記
- **文体**: 常体（だ・である調）を使用してください
- **固有名詞**: 原語のカタカナ表記を優先してください（例: Mark Twain → マーク・トウェイン）
- **文化的背景**: 直訳を避け、日本語読者に自然な表現を使ってください
- **段落構造**: 原文の段落分けを尊重してください

### 品質基準
- 自然で読みやすい日本語表現を心がけてください
- 原文の意味を正確に伝えてください
- 固有名詞の表記を統一してください
- 文字化けや記号の異常がないように注意してください

## 出力形式

以下のJSON形式で出力してください：

```json
{
  "translated_text": "翻訳された本文をここに記載してください。段落の改行は維持してください。",
  "summary": "このパートの要約を200文字以内で記載してください。",
  "keywords": ["キーワード1", "キーワード2", "キーワード3"]
}
```

### 出力の注意事項
- `translated_text`: 原文の全内容を翻訳してください。省略や要約ではなく、完全な翻訳が必要です
- `summary`: このパートの内容を簡潔にまとめてください（200文字以内）
- `keywords`: このパートの重要なキーワードを3〜5個選んでください（日本語）

必ずJSON形式のみを出力してください。説明や追加のコメントは不要です。
